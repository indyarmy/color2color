colorcolor Javascript Library

Version 3.0.1, Released 2022/11/24

MIT, Copyright 2022 Russel Porosky

Description

colorcolor converts a color string in these CSS formats (HEX, HEXA, HSB, HSL, HSLA, HSV, HWB, RGB, or RGBA) into another format (also HEX, HEXA, HSB, HSL, HSLA, HSV, HWB, RGB, or RGBA).

Install

colorcolor is available via NPM:

npm install colorcolor

Or download the minified distribution file from Github instead if you prefer :)

Usage

colorcolor(originalColor: string, [newColorType: string(hex|hexa|hsb|hsl|hsla|hsv|hwb|rgb|rgba) = 'rgba'])

The first argument is the original color string in HEX, HEXA, HSB, HSL, HSLA, HSV, HWB, RGB, or RGBA format.

The second argument (optional) is which format you'd like the new color string to be in. The default is 'rgba'.
